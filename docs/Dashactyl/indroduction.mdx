# Introduction

Dashactyl is a client area dashboard for [Pterodactyl](https://pterodactyl.io) API that allows for users to view and manage their resources for their servers. You can also set limits on how many servers and resources that are given to users.

If you encounter any issues installing Dashactyl, [join the Discord](https://discord.gg/nDxrPWh8pk) and use one of the dedicated support channels to ask your question.
