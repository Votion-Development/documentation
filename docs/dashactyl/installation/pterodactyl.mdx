
# Installation via Pterodactyl

Dashactyl provides an egg for the Pterodactyl panel to host it on.

<a href="/files/egg.json" className="button button--secondary button--md" download="dashactyl-egg.json">
<>Download Egg File</>
</a>

## Installation

Log into your Pterodactyl instance, and go onto the Admin panel.

You'll then want to go to Nests and press the green button named **"Import Egg"**

Choose the json file you just downloaded, and choose whatever nest you choose.
